# Claude Code 사용법

## Claude Code란?
터미널에서 실행되는 AI 어시스턴트로, 파일을 직접 읽고 쓰고 수정할 수 있습니다.

---

## 설치 및 인증 확인

| 명령어 | 설명 |
|--------|------|
| `claude --version` | 설치된 버전 확인 |
| `claude doctor` | 설치 상태 진단 (모든 항목 체크 표시 = 정상) |

---

```bash
cd ~/Documents/Obsidian/Retirement-Seminar-Self-Study
claude
```

## 주요 명령어

### 대화 중 사용하는 명령어

| 명령어 | 기능 |
|--------|------|
| `/help` | 도움말 보기 |
| `/init` | CLAUDE.md 프로젝트 설명서 생성 |
| `/config` | 설정 확인 및 변경 |
| `/clear` | 대화 내역 초기화 |
| `/output-style explanatory` | 교육용 설명 모드 전환 |

### 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Shift+Tab` | Plan 모드 전환 (계획 먼저 세우기) |
| `Ctrl+C` | 현재 작업 중단 |
| `Esc` | 대화 종료 |

## CLAUDE.md 파일의 역할

`/init` 명령어로 생성되는 `CLAUDE.md`는 **프로젝트 설명서**입니다.
- Claude Code 실행 시 자동으로 읽어서 프로젝트 맥락을 파악
- 잘 작성해두면 매번 처음부터 설명할 필요 없음
- "AI에게 주는 업무 인수인계서"라고 생각하면 됩니다

## 팁

1. **구체적으로 요청하기**: "파일 수정해줘" 보다 "README.md의 3번째 항목을 수정해줘"가 더 정확한 결과를 얻습니다
2. **Plan 모드 활용**: 복잡한 작업은 `Shift+Tab`으로 Plan 모드에서 계획을 먼저 세운 후 실행하세요
3. **CLAUDE.md 관리**: 프로젝트가 변경되면 CLAUDE.md도 업데이트하세요
